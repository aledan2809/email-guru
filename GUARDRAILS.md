# GUARDRAILS — E-mail Guru

## Architecture
- No new frameworks without ADR approval
- No database schema changes without explicit request
- No removal of existing API endpoints

## Code
- No force push to main/production branches
- No deletion of files without explicit approval
- No modification of .env files without explicit approval
- No changes to CI/CD pipeline without explicit approval

## Deploy
- No production deploy without explicit approval
- No DNS/domain changes without explicit approval
- Rollback plan required before any production change
